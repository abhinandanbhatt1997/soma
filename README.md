Soma is a small, interpreted functional programming language inspired by **lambda calculus**.  
It supports **lambdas**, **if expressions**, **arithmetic operations**, and **function calls**.

---

## Features

- Anonymous functions (lambda expressions)
- `if-then-else` expressions
- Arithmetic: `+`, `-`, `*`, `/`
- Variables and assignments
- Function calls
- Comments with `#`
- Interactive REPL
